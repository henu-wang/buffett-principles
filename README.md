# Warren Buffett Investment Principles Explorer

An interactive single-page application exploring 20 key investment principles from Warren Buffett, the Oracle of Omaha. Built with vanilla JavaScript and Tailwind CSS.

**Live Site:** [https://henu-wang.github.io/buffett-principles/](https://henu-wang.github.io/buffett-principles/)

## Features

- **20 Core Principles** - Each with Buffett's original quote, detailed explanation, real-world example, and actionable application tips
- **Category Filtering** - Browse by Value Investing, Risk Management, Business Quality, Market Psychology, or Personal Growth
- **Full-Text Search** - Instantly find principles by keyword across titles, quotes, and explanations
- **Quiz Mode** - Test your knowledge with 10 randomized questions drawn from a pool of 15
- **Principle Detail View** - Expandable modal with complete principle breakdown and navigation
- **Dark Theme** - Elegant Berkshire Hathaway-inspired design with gold accents
- **Mobile Responsive** - Optimized layout for all screen sizes
- **SEO Optimized** - Full meta tags, Open Graph, Twitter Cards, and JSON-LD structured data

## Development

This is a single-file application. To develop locally:

```bash
# Clone the repository
git clone https://github.com/henu-wang/buffett-principles.git
cd buffett-principles

# Open in browser
open index.html
```

No build step required. Edit `index.html` and refresh.
